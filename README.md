# 甜甜育儿记录本 (Tiantian Cat Care Diary)

一个温馨治愈风格的宠物育儿记录单页Web应用，专为记录母猫和幼崽的成长日常而设计。

## ✨ 功能特性

### 🏠 首页 Dashboard
- 母猫甜甜信息卡（头像、名字、年龄、品种、体重，可修改）
- 今日概览（喂食次数、喂药次数、吃奶次数、照片数量）
- 快捷按钮（记录喂食、记录喂药、记录吃奶、上传照片）
- 时间轴（按时间展示所有最近记录）

### 🐱 幼崽管理
- 5只幼崽：奶油崽、灰灰、团子、奶糖、小白
- 每只幼崽详细信息：编号、名字、性别、毛色、出生日期、体重
- 幼崽详情页：成长记录、体重曲线、成长记录时间轴

### 🍽️ 喂食记录
- 食物类型：幼猫粮、猫罐头、鸡胸肉、羊奶粉、自定义
- 食用量（g/ml）
- 备注、照片上传

### 💊 喂药记录
- 药品名称、剂量、单位（片/ml/滴/mg）
- 备注、照片上传

### 🍼 幼崽吃奶记录
- 开始/结束时间，自动计算时长
- 参与幼崽多选（支持全选）
- 备注、照片上传

### 📸 照片管理
- 瀑布流展示
- 分类查看（甜甜、喂食、喂药、幼崽成长）
- 点击放大查看
- 图片自动压缩

### 📅 日历记录
- 月视图
- 每天显示记录标记
- 点击日期查看当天所有记录

### 📊 统计分析
- 喂食趋势（近7天柱状图）
- 喂药统计（饼图）
- 吃奶频率（近7天折线图）
- 幼崽体重增长曲线

### 🔔 提醒功能
- 喂食提醒、喂药提醒、吃奶提醒
- 支持每天固定时间或间隔小时提醒
- Notification API + 浏览器内弹窗提醒

### ☁️ 云端同步
- Supabase 云端数据库
- 多设备数据同步，家人共享
- 本地缓存，离线可用
- 家庭码数据隔离
- 自动/手动同步

### ⚙️ 设置模块
- 深色/浅色模式切换
- 数据导出/导入（JSON备份）
- 数据清除
- 通知权限管理
- 云端同步配置

## 💾 数据存储

所有数据存储在浏览器 localStorage 中，包含以下数据表：

- `mother_cat` - 母猫信息
- `kittens` - 幼崽列表
- `feeding_logs` - 喂食记录
- `medicine_logs` - 喂药记录
- `nursing_logs` - 吃奶记录
- `photos` - 照片数据（base64）
- `reminders` - 提醒设置
- `settings` - 用户设置

## 🎯 使用说明

### 基础使用
1. **首次使用**：打开页面后会自动初始化默认数据
2. **添加记录**：点击首页快捷按钮或各页面的添加按钮
3. **查看数据**：通过底部导航切换不同功能页面
4. **备份数据**：在设置页面导出JSON备份文件
5. **恢复数据**：在设置页面导入之前的备份文件

### 云端同步（可选）
如需与家人共享数据，可配置云端同步：
1. 创建 Supabase 项目（参考 [SYNC_SETUP.md](./SYNC_SETUP.md)）
2. 执行数据库 Schema 创建数据表
3. 在设置 → 云端同步中填写配置信息
4. 设置家庭码，家人使用相同家庭码即可共享数据

详细配置步骤请查看 [云端同步配置指南](./SYNC_SETUP.md)。

## 💜 温馨提示

- 数据存储在浏览器本地，清除浏览器数据会导致记录丢失，请定期导出备份
- 照片会自动压缩以节省存储空间
- 提醒功能需要浏览器保持打开状态才能正常工作
- 建议添加到手机桌面，获得更好的使用体验
- 云端同步为可选功能，不配置也可以正常使用本地模式
